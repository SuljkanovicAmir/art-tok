# Favicon & App Icon

**Date:** 2025-03-25

## Changes

- Created `public/favicon.svg` — minimalistic picture frame with landscape (mountains + sun) on dark #050505 background, optimized for 16/32px browser tabs
- Updated `public/icons/icon.svg` — same design scaled for 512px PWA home screen icons
- Updated `index.html` — favicon points to new art icon instead of default Vite logo
- Removed `public/vite.svg` — no longer needed

## Gotcha

Vite automatically prepends the `base` path (`/art-tok/`) to href values in `index.html`. Do NOT hardcode `/art-tok/` in href — use root-relative paths like `/favicon.svg`.
